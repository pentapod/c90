React Native で
アプリ、作ってみませんか？

React Native とは、Facebookが開発を行っているスマートフォンアプリ開発用のUIフレームワークです。 React Nativeを使えば、フロントエンド界の有名ライブラリReact.jsを使って書くのと変わらない感覚で、アプリ開発を行うことができます。 しかも、内部ではOS標準のUIの管理を行っているので、ネイティブのアプリと全く見分けがつきません。 本書には、そんなReact Nativeに入門するまでの内容が揃っています。 「はじめる React Native」を読んで、あなたもアプリ開発をはじめませんか？

こんな人にオススメです

  • JavaScriptはそれなりに使えるけど､アプリ開発はさっぱり
  • React.jsに興味がある
  • JavaとSwiftどっちも覚えるなんて面倒だ
